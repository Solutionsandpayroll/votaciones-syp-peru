# Sistema de Votación - SYP Perú

Sistema profesional de votación desarrollado en React que permite a los usuarios autenticarse y votar por su candidato preferido.

## Características

- 🔐 Autenticación de usuarios con nombre completo
- 🗳️ Interfaz intuitiva para votación
- 📊 Visualización de resultados en tiempo real
- 🎨 Diseño moderno y responsivo
- ✅ Confirmación de voto
- 🔒 Prevención de voto duplicado

## Tecnologías

- React 18
- React Router para navegación
- Vite como bundler
- CSS moderno con variables y flexbox/grid

## Instalación

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build
```

## Estructura del Proyecto

```
src/
├── components/       # Componentes reutilizables
├── pages/           # Páginas principales
├── context/         # Estado global de la aplicación
├── data/            # Datos de candidatos
├── styles/          # Estilos CSS
└── main.jsx         # Punto de entrada
```

## Uso

1. Ingresar con nombre completo
2. Ver lista de candidatos
3. Seleccionar candidato preferido
4. Confirmar voto
5. Ver resultados
